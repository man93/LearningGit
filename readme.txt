1-) Pdf_merger.py------

a-)this script takes two pdf files on command line(in the current directory) and merges these  two in one pdf file(you can give name to this pdf file too).

2-) Pdf_merger_all.py------
a-) this script merges all pdf files in the current directory and can merge pdf files in the given directory path.

3-) Pdf_cutter.py
a-) this script take pdf file as input and page numbers to be cut from the pdf file and give  a new pdf file after deleting the given pages.

4-)Pdf_encrypter.py---
a-) this script merges all the pdf files in the current as well as given directory path and encrypts the final pdf with given password.
